## ğŸš€ Getting Started

### Prerequisites
- Node.js (v16+)
- npm or yarn
- MongoDB Atlas account (or local MongoDB)

### Installation

1. Clone the repository
```bash
git clone https://github.com/yashvi-3106/focus_fuze.git
cd focus_fuze
```

2. Backend Setup
```bash
cd Backend
npm install
cp .env.example .env
npm run dev
```

3. Frontend Setup
```bash
cd ../Frontend
npm install
cp .env.example .env
npm run dev
```

4. Open your browser
- Frontend: http://localhost:5173
- Backend API: http://localhost:3000
